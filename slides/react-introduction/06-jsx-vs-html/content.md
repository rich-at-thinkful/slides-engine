# JSX vs HTML

```js
// HTML
<img src="doggo.jpg" alt="Good boy">
// JSX - must close all tags
<img src="doggo.jpg" alt="Good boy" />
```

```js
// HTML
<div class="doggo selected"></div>
// JSX - className instead of class
<div className="doggo selected"></div>
```

```js
// HTML
<label for="username">Username:</label>
<input id="username" type="text">
// JSX - htmlFor instead of for
<label htmlFor="username">Username:</label>
<input id="username" type="text" />
```
