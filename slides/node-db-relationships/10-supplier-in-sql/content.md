# Supplier table in SQL

<div class='row'>
<div class='cell-4'>

```sql {#supplier-sql-1}
DROP TABLE IF EXISTS supplier;

CREATE TABLE supplier (
  id INTEGER PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  supplier_name TEXT NOT NULL,
  phone TEXT,
  city TEXT
);
```

</div>
<div class='cell-2 smallest'>
