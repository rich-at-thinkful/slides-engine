# Connecting External JavaScript To HTML

index.html
```
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>JavaScript on your page</title>
  </head>
  <body>
    <h1>Welcome to my website</h1>
    <script src="index.js"></script>
  </body>
</html>
```
